# DayZ Automation & High-Performance Scripts

Este projeto é uma solução completa de automação e otimização para **DayZ**, combinando o poder do desenvolvimento mobile com scripts de baixo nível para hardware de alta performance. O objetivo é fornecer uma experiência de jogo mais fluida, técnica e personalizada.

## 🌟 Sobre o Projeto

O projeto nasceu da necessidade de integrar ferramentas de automação acessíveis com o controle preciso exigido em cenários competitivos de DayZ. Ele se divide em dois pilares principais:

1.  **Ecossistema Mobile (Android):** Um aplicativo desenvolvido em Kotlin que atua como o cérebro das macros, utilizando serviços de sobreposição (Overlay) e persistência de dados em nuvem via Firebase.
2.  **Scripts de Hardware (GPC):** Implementações otimizadas para dispositivos de precisão, focadas em controle de balística, movimentação avançada e interface de usuário intuitiva diretamente no hardware.

##  Objetivos Principal

- **Automação Inteligente:** Facilitar o gerenciamento de macros e coordenadas complexas através de uma interface flutuante.
- **Acessibilidade:** Interface totalmente em Português (PT-BR) para facilitar a configuração em tempo real.
- **Eficiência de Código:** Scripts limpos e otimizados para evitar gargalos de memória e garantir a máxima estabilidade durante o gameplay.
